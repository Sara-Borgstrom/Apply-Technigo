METADATA

INFO

Metadata is information about information. According to wikipedia metadata is defined as the data providing information about one or more aspects of the data; it is used to summarize basic information about data which can make tracking and working with specific data easier. Some examples include:

  • Means of creation of the data
  • Time and date of creation
  • Creator or author of the data
  • Location on a computer network where the data was created
  • Standards used
  • File size
  • Data quality
  • Source of the data
  • Process used to create the data
